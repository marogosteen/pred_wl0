# predwl0

## Description
液性限界の予測するNeural Networkを訓練します．

## Getting Started
パッケージのインストール
```
pip install -r requirements.txt
```

pytrochのインストールは実行環境によって違うため，公式のドキュメントからインストールコマンドを取得してください．

[PyTorch公式ページ](https://pytorch.org/)

## Executing program
学習の実行
```
python main.py
```

## MAEの表示

python show_mae.py
